# Chat.VK Frontend

React + Vite frontend for Chat.VK application.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

The frontend will run on `http://localhost:3000`

## Build

To build for production:
```bash
npm run build
```

## Project Structure

- `/src/pages` - Main page components (Login, Signup, Chat)
- `/src/components` - Reusable components
- `/src/styles` - CSS stylesheets
```
